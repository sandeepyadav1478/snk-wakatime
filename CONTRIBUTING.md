# Contributing

This is a personal fork of [Platane/snk](https://github.com/Platane/snk) with WakaTime integration.

Bug reports and issue discussions are welcome. Pull requests are generally not accepted, but feel free to open an issue to discuss changes first.

For contributions to the core snake algorithm, please consider contributing to the [upstream project](https://github.com/Platane/snk) instead.
